# 🌸 Akazuba Florist Admin Dashboard Guide

## 🎯 Overview

Your admin dashboard is now fully functional and allows you to manage every aspect of your flower business without touching any code. Everything can be done through the user interface!

## 🚀 Accessing the Admin Dashboard

1. **Login**: Go to `/admin` on your website
2. **Credentials**: Use your admin username and password
3. **Security**: The dashboard is protected and only accessible to authorized users

## 📊 Dashboard Sections

### 1. **Overview Tab**
- **Sales Analytics**: View total sales, revenue trends, and growth metrics
- **Order Statistics**: Track order counts and status distribution
- **Customer Insights**: Monitor customer growth and engagement
- **Product Performance**: See which products are selling best

### 2. **Products Tab** 🔥 **NEW FEATURES**

#### **Add New Product**
1. Click "Add Product" button
2. Fill in product details:
   - **Name**: Product name (e.g., "Premium Red Roses")
   - **Category**: Select from Roses, Tulips, Lilies, Mixed Bouquets, etc.
   - **Price**: Enter price in RWF (e.g., 25000)
   - **Stock**: Available quantity
   - **Status**: Active, Inactive, or Out of Stock
   - **Description**: Detailed product description

#### **Image Upload** 🖼️
- **Drag & Drop**: Simply drag images into the upload area
- **Multiple Images**: Upload several images per product
- **Preview**: See all uploaded images before saving
- **Remove**: Click the X to remove unwanted images
- **File Types**: Supports JPG, PNG, JPEG
- **Size Limit**: Up to 5MB per image

#### **Edit Products**
- Click the edit icon (pencil) next to any product
- Modify any details including images
- Add or remove images
- Update prices, stock, and descriptions
- Save changes instantly

#### **Search & Filter**
- **Search**: Find products by name or description
- **Category Filter**: Filter by product category
- **Export**: Download product data
- **Import**: Bulk upload products

### 3. **Orders Tab**
- **View All Orders**: See complete order history
- **Order Details**: Click any order for full information
- **Status Updates**: Change order status (Pending → Processing → Delivered)
- **Customer Info**: Access customer details and contact information
- **Export Orders**: Download order data for record keeping

### 4. **Customers Tab**
- **Customer List**: View all registered customers
- **Customer Details**: See order history and preferences
- **Analytics**: Track customer growth and engagement
- **Search**: Find specific customers quickly

### 5. **Settings Tab**
- **General Settings**: Shop name, contact info, business hours
- **Payment Settings**: MoMo and BK account details
- **Account Management**: Update admin credentials

## 🆘 Troubleshooting

### **Common Issues**

**Can't Upload Images?**
- Check file size (max 5MB)
- Ensure file type is JPG, PNG, or JPEG
- Try refreshing the page

**Changes Not Saving?**
- Check internet connection
- Try clicking save again
- Refresh the page and retry

**Can't Access Admin?**
- Verify login credentials
- Clear browser cache
- Try different browser

### **Support**
If you encounter any issues:
1. Check this guide first
2. Try refreshing the page
3. Clear browser cache
4. Contact technical support
