# Simplified Admin Panel

This is a streamlined admin panel for the Akazuba Florist e-commerce store, designed to be simple and easy to use.

## Features

### 🏠 Dashboard
- Overview of key metrics (orders, products, revenue, customers)
- Recent orders display
- Quick action buttons
- System status indicators

### 📦 Products Management
- View all products in a clean grid layout
- Add new products with image upload
- Edit existing products
- Delete products
- Search and filter products
- Category management

### 🛒 Orders Management
- View all orders with status tracking
- Update order status (Pending → Processing → Shipped → Delivered)
- Order details view
- Customer information display
- Payment status tracking

### ⚙️ Settings
- Store information configuration
- Financial settings (currency, tax rate, delivery fees)
- Business hours and contact details

## Navigation

The admin panel has a simple sidebar navigation with only 4 main sections:
1. **Dashboard** - Overview and quick actions
2. **Products** - Manage your flower and perfume inventory
3. **Orders** - Track and manage customer orders
4. **Settings** - Configure store settings

## Key Improvements

- Core e-commerce functions only

- Error handling and loading states

## Usage

1. **Login** - Use your admin credentials to access the panel
2. **Dashboard** - Check your store overview and recent activity
3. **Add Products** - Use the "Add Product" button to create new listings
4. **Manage Orders** - Update order status as they progress
5. **Configure Settings** - Set up your store information and preferences

## Technical Details

- Built with Next.js 14 and React
- Uses Tailwind CSS for styling
- Responsive design for mobile and desktop
- Image upload functionality
- Real-time status updates
- Form validation and error handling

## Support

If you encounter any issues with the admin panel, check:
1. Your internet connection
2. Browser compatibility (Chrome, Firefox, Safari, Edge)
3. Clear browser cache if needed
4. Contact support if problems persist

**Note**: This simplified admin panel focuses on the core functionality needed to run your flower and perfume store efficiently. All complex features have been removed to ensure stability and ease of use.
